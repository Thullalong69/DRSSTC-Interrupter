# DRSSTC Interrupter

Raspberry Pi–based interrupter for DRSSTC operation.

## Features
- Web interface built with Flask
- Hardware-based pulse skip support
- Adjustable BPS (bursts per second)
- Burst mode
- T_ON / T_OFF mode
- Single-shot mode
- Jitter-free interrupter signal using pigpio hardware PWM
- MIDI file playback with a large library of converted MIDI files

⚠️ **Warning:** This is a high-voltage project. Use at your own risk.
